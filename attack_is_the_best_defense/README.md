Optional project on network security; eg network sniffing and ARP spoofing using Docker

Tools:
- wireshark
- telnet
- tcpdump
- docker
- hydra

Tasks:
0-sniffing: Using tcpdump cmd, sniff the provided network to determine the user's password

1-dictionary_attack: Using Docker, pull the given image and using hydra cmd, brute forcefully overload the account to determine the password of the user account via ssh on docker container
